Dead Moon

Dead Moon is a group project using python and pygame in where you play a top down zombie survivor shooter on an abandoned space station on the moon.

Software Used

  • Python
  • PyGame
  • GitHub

Premise

Dead Moon was a end of year freashman game project. The goal was to create a small scale top down zombie shooter in pygame and to colaborate with others using GitHub.

Responsibilities

In this colaboration project I worked on creating the base character, the Player movement and controls, the base zombie and other zombie types including the fast armless zombie, the giant slow zombie, and the shooting zombie. I also created the base weapon and projectile as well as the most of the Player's HUD including weapon selection, health, cash, and ammo counter. I also created a reusable healthbar class that can be added to any other gameobject and displays it's health.

Some Features

Enemy Types: We had four enemies types, grunt zombie, tank zombie, runner zombie, shooter zombie. For both the runner and the shooter zombie edited the sprites to make custom animations for each of them.

Weapon Types: We created multiple ranged weapons for this project that each behave differently. From the projectile spread of the Shotgun to the high damage of the Sniper. Other weapons included the pistole, assult rifle, RPG launcher, and the Raygun. The Player could collect these weapons and cycle through them one at a time.
